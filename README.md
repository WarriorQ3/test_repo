## Инструкция по запуску

1. Скопируйте себе локально репозиторий (предпочтительный способ - предварительно установить git)

   git clone https://github.com/WarriorQ3/my_repo.git

   или откройте репозиторий c вышеуказанного сайта через GitHub Desktop

   или скачайте с вышеуказанного сайта и распакуйте себе локально zip-архив 
2. Соберите, создайте и запустите контейнеры, находясь в папке проекта (предварительно установите docker согласно инструкций https://docs.docker.com/engine/install/)

   sudo docker compose up -d
3. Желательно проверить, что все контейнеры запустились
   docker compose ps

## Проверка работоспособности

1. **Основная проверка доступности на порту 80**

   curl http://localhost
   Ожидаемый ответ: "Hello from Effective Mobile!"
2. **Проверка изоляции HTTP-сервера на Python**

   curl http://localhost:8080

   Вместо порта 8080 можно проверить доступность любых других портов
   Ожидаемый ответ: "Could not connect to server"
   
## Краткое описание архитектуры проекта

Сервисы nginx и backend работают в отдельных контейнерах и взаимодействуют через внутреннюю сеть bridge. 

Nginx принимает запросы с хоста c порта 80 и проксирует их на backend на порт 8080.

## Использованные технологии

1. **Docker** — контейнеризация приложений
2. **Docker Compose** - оркестрация приложений
3. **Python 3.11** (alpine) — HTTP-сервер, реализованный на языке python
4. **Nginx** — обратный proxy-сервер
